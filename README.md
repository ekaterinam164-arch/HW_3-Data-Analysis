# Анализ данных на материале датасета «Художественная проза о крестьянах в Российской империи 1772‒1861»

Данные были взяты отсюда: https://dataverse.pushdom.ru/dataset.xhtml?persistentId=doi:10.31860/openlit-2024.11-B018

Датасет представляет собой библиографические данные о 235 сочинениях с протагонистами-крестьянами, 
написанных в Российской империи до отмены крепостного права. Внутри находятся столбцы со следующими значениями

1) Номер
2) Автор
3) Псевдоним
4) Заголовок
5) Подзаголовок
6) Источник —— первая публикация произведения
7) Город издания
8) Том / номер / часть / даты выпуска и/или пагинация текста в отдельных изданиях
9) Издания до 1861 г.
10) Тип наррации
11) Сеттинг
12) Регион действия
13) Место действия — название конкретного населенного пункта, где происходит действие
14) Пол главного героя/героини
15) Имя мужского протагониста
16) Имя женской протагонистки
